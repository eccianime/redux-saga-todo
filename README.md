### Code Challenge

For this test, I created a simple Redux Saga controlled Todo List for handling side-effects.

### Installation

Just clone the repo with

    git clone https://github.com/eccianime/redux-saga-todo.git

and install with

    npm install

or

    yarn

### Running the app

Go to the folder with

    cd redux_saga_todo

and run

    npm run start

or

    yarn start
